Error Management
==========================
.. include:: ../definitions.rst

It is recommended to first read :ref:`The Fundamentals <fundamentals>` to
get up-to-date with the fundamentals behind the |pylib_name| package.
There the different layers of the |pylib_name| hierarchy are introduced. 
We here go through each layer and show the different types of errors that could 
occur and how to debug them. Before doing so, we illustrate how to get access 
to a verbose serialization mode.
